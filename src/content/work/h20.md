Developed a high-fidelity landing page and interactive dashboard for SolToken, a simulated Solana-based cryptocurrency project. The goal was to create a professional-grade web3 interface that demonstrates seamless wallet connectivity and real-time data visualization.

I implemented custom animations and dynamic data sections to mimic real-time blockchain activity. I overcame significant UI/UX challenges by refining complex CSS layouts and ensuring cross-browser compatibility for advanced glassmorphism and gradient effects common in the crypto space.

Key Technical Challenges Overcome:

-Wallet Integration Complexity: Implemented secure wallet connection flows (Phantom/Solflare) using @solana/web3.js, managing connection states and handling edge cases like user-side transaction cancellations.

-Real-time Data Sync: Optimized the dashboard to fetch and display simulated market data without heavy re-renders, utilizing useMemo and useCallback to maintain a 60 FPS interface during price fluctuations.

-State Management: Architected a centralized state using React Context (or Zustand) to handle complex user data, including token balances and transaction history, across multiple modular components.

-Responsive UX Design: Built a mobile-first responsive layout that ensures the complex data-heavy dashboard remains readable and functional on small screens.
